Privacy Policy

Last updated: May 13, 2026

Humanify ("the Extension") is committed to protecting your privacy. This policy explains what data we handle and how.

1. Data We Collect

We do NOT collect personal data. The Extension does not track browsing history, collect analytics, or store any personally identifiable information on remote servers.

  • Text you submit for rewriting: Sent only to the AI provider (DeepSeek API) for processing. Not stored on our servers.
  • API Key (optional): If you provide your own DeepSeek API key, it is stored locally in Chrome's chrome.storage.local and never leaves your device except when sent directly to DeepSeek for authentication.
  • Usage counter: A daily usage count is stored locally to enforce the free tier limit (3 rewrites/day). This data never leaves your device.

2. How We Use Data

The text you select and submit for rewriting is transmitted to DeepSeek API (api.deepseek.com) solely for the purpose of generating a rewritten version. DeepSeek's privacy policy governs their handling of that data.

No data is stored on any server we control. All processing is transient.

3. Third-Party Services

The Extension uses one third-party service:

  • DeepSeek API — AI text rewriting. See DeepSeek Privacy Policy.

No analytics, advertising, or tracking SDKs are included.

4. Data Storage

All local data (API key, usage count) is stored in Chrome's chrome.storage.local. You can clear this at any time via Chrome Settings -> Privacy -> Clear browsing data, or by uninstalling the Extension.

5. Your Rights

  • Access: You can inspect all stored data via Chrome DevTools (Application -> Storage -> chrome.storage.local).
  • Delete: Uninstalling the Extension removes all local data immediately.
  • Opt out: Simply stop using the Extension. No data persists beyond what is on your device.

6. Permissions Explained

  • activeTab: Needed to read the selected text on the current page for rewriting.
  • contextMenus: Needed to add "Rewrite with Humanify" to the right-click menu.
  • storage: Needed to save your API key and daily usage count locally.
  • scripting: Needed to inject the rewriting interface into the page.
  • commands: Needed for the Ctrl+Shift+R keyboard shortcut.

7. Children's Privacy

The Extension is not directed at children under 13 and does not knowingly collect data from them.

8. Changes to This Policy

Updates will be reflected on this page. The "Last updated" date will change accordingly.

9. Contact

For questions about this privacy policy, open an issue at GitHub Issues.
